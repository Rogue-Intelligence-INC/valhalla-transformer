# Valhalla 与 Transformer：简明简介

**版本**: 1.1 · **日期**: 2026-06-16  
**融资 MVP**: [05_融资MVP实验报告.md](./05_融资MVP实验报告.md) · **Rogue Intelligence LNC.**

---

## 我们是什么

**Valhalla** 是一套可私有化部署的 **AI 孵化运行时**：把文本/语料编码为 f64 信号，经 **Hub（路由）· Tile（块记忆）· StemCell（自组织结构）· Fate（收敛协调）** 多轮演化，产出可观测的结构状态，再用于 **改变模型行为** 或 **原生解码答案**。

**Transformer** 是当前主流「语言智能」实现：大规模预训练 + 自注意力 + 梯度微调。Valhalla **不否认** Transformer 在通用推理上的优势；我们提供的是 **另一条路径**——用 **结构生长与场收敛** 替代「只靠 backprop 改权重」作为 **垂直领域 AI 孵化** 的核心机制。

---

## 两层架构：Tier A 与 Tier B

```
┌─────────────────────────────────────────────────────────────┐
│  Tier B（终局）— Valhalla 即计算本体，不加载 Transformer     │
│  语料 + 问题 → Ingress → Hub/Tile/StemCell → 原生解码       │
└─────────────────────────────────────────────────────────────┘
                              ↑ 路线
┌─────────────────────────────────────────────────────────────┐
│  Tier A（当前代理）— 结构 export → patch 外部小模型 (0.5B)    │
│  语料 → Valhalla → patch → Qwen generate → 测 Δacc          │
└─────────────────────────────────────────────────────────────┘
```

| | Tier A | Tier B |
|--|--------|--------|
| **计算主体** | Qwen2.5-0.5B Transformer | Hub + Tile + StemCell + Fate |
| **孵化动作** | 结构 patch 权重 | 结构演化 + 检索/规则解码 |
| **已跑通** | ✓ 48 题 strict 实验 | ✓ `valhalla_native_qa` 48 题 |
| **当前 acc** | baseline ~69%，patch 后 **无净提升** | baseline ~23%，triad **25%** |

## 2026-06-16 实验结论（诚实版）

**基准**: 48 题 × 13 域（GSM8K、MMLU、开放问答、中文等），**strict MCQ** 评分。

### Tier A — Patch 外部 Transformer

| 方法 | Δacc（strict） | 含义 |
|------|----------------|------|
| Valhalla triad patch | **-2.08 pp** | 最优 Valhalla 臂，仍略降 |
| prompt_only（语料进 prompt） | -6.25 pp | 语料 alone 不够 |
| random_patch（同强度随机） | **-14.58 pp** | Valhalla **显著优于随机** |
| logit_kd（3B→0.5B 蒸馏） | **+22.92 pp** | 有梯度监督的上限参照 |

**可对外说**: Valhalla 结构 patch **确实改变输出**（变化率 ~90%），且 **不是任意噪声**；但在 strict 条件下 **尚不能** 宣称提升小模型 benchmark 准确率。

### Tier B — 原生（200Q 融资 MVP）

| 指标 | 值 |
|------|-----|
| 整体 acc | 24.50% vs Transformer **68%** |
| **数值子域** | **61% vs 71%** |
| 语料 Δ | 0 pp（CI ±2） |

**48Q 的 +2.08 pp 在 200Q 上消失** — 融资叙事应以 200Q 为准。

**可对外说**: 原生路径 **已端到端跑通**；数值子集有信号；**整体仍远低于 Transformer**，不能宣称已替代。

---

## 与 Transformer 的关系（定位句）

> **Valhalla 是面向「可拥有、可审计、可结构孵化」的 AI 运行时；Transformer 是现阶段通用语言能力的强基座。Tier A 把 Valhalla 结构「嫁接」到 Transformer 上验证可触达性；Tier B 逐步让 Valhalla 自身承担推理，最终目标是 AI 孵化平台而非 API 套壳。**
