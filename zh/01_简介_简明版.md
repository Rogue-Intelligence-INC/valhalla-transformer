# Valhalla 与 Transformer：简明简介

**版本**: 1.3 · **日期**: 2026-06-16  
**可视化**: `viz/` · **测试 API**: `api/` · **Rogue Intelligence LNC.**

---

## 我们是什么

**Valhalla** 是私有化 **AI 孵化运行时**：文本 → f64 信号 → **Hub / Tile / StemCell** 结构演化 → 改变模型行为（Tier A）或 **原生解码**（Tier B）。

**Transformer** 是通用语言智能的强基座。Valhalla **不声称**整体替代它；我们提供 **结构孵化** 另一条路径。

| 维度 | **进步了** | **没进步 / 退步了** |
|------|-----------|---------------------|
| **实验范式** | 200Q + body 分离 + CI + Transformer 对标 | — |
| **工程闭环** | Tier B `valhalla_native_qa` 无 HF 端到端 | — |
| **结构信息** | Tier A：triad patch 优于 random（-2.08 vs -14.58 pp） | Tier A strict **无净 acc 提升** |

| **Body 可分离** | hub/tile/stem/triad 各 200Q 跑通 | 单 body **23.5%**，比 triad -1pp |
| **语料孵化** | 200Q 诚实披露 Δ **0pp**（CI±2） | 48Q +2.08pp **作废**（小样本噪声） |

> **一句话**：范式与证据链在进步；**通用 benchmark 能力**尚未进步，不能宣称替代 Transformer。

## Hub / Tile / StemCell 内部发生了什么

语料每行经 f64 编码后进入 `TriadSession::with_body(...)`。**Body 决定哪几路子系统参与 ingress**：

### Hub（调度中枢）

- 四元路由 **BH / MB / … / BF** → 统计 `hub_routed[4]`
- Fate 收敛后输出 **`hub_prefs`**（4 维亲和力）
- **Tier B**：`hub_prefs` **加权** memory 检索与 MCQ 选项打分
- **单开 Hub**：MCQ 翻转最多（Fate 权重改变选项排序）→ 200Q **-1pp**

### Tile（俄罗斯方块）

- 标量流 → 块 **merge / split** → `completed_tiles`
- 每块导出 **`tile_signatures`**（f64 向量）
- **1-cycle 收敛**（c5/c10 = c1）
- **Tier B**：tile 签名参与 numeric / cortex 候选加权
- **单开 Tile**：numeric 不变；MCQ 略降

### StemCell（干细胞图）

- 细胞 **merge / divide** → 按 energy 取 top-32 → **`stem_signatures`**
- 记录 `stem_connections`
- **Tier B**：stem 签名参与 numeric 加权
- **单开 Stem**：效应类似 Tile

### Triad（三体同时）

- Hub + Tile + StemCell **并行 feed**，融合为 **`patch_vector`（256 维）**
- MCQ 路由与 baseline（triad 无语料）**一致** → 200Q **24.5% ≥ 单 body 23.5%**
- **numeric 四臂完全相同**（61.02%）— 算式 decode 不依赖 body 差异

```
语料行 → encode → [Hub 路由 + Tile 块 + Stem 细胞] → Fate
         → patch_vector / hub_prefs / tile_signatures / stem_signatures
         → Tier A: patch Qwen  |  Tier B: decode_numeric | mcq_option | open_retrieval
```

## 两层架构

| | Tier A（代理） | Tier B（原生） |
|--|----------------|----------------|
| 计算 | Qwen2.5-0.5B + Valhalla patch | Hub+Tile+Stem+Fate |
| 48Q strict | patch 后 **无净提升** | triad **25%**（200Q **24.5%**） |
| 可陈述 | 结构 **≠** 随机噪声 | 数值楔子；**不能替代** 68% TF |
