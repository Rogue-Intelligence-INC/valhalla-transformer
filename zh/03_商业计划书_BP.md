## 执行摘要

Rogue Intelligence 构建 **Valhalla**——可私有化部署的 **AI 孵化运行时**，用 **Hub / Tile / StemCell / Fate** 结构演化替代「只靠 Transformer + backprop」作为垂直领域 AI 的 **第二条路径**。

**2026-06 实验结论（诚实）**:
- **Tier A**（结构 patch Qwen-0.5B）：strict 48 题 **无净 acc 提升**，但 **优于 random patch 12.5 pp**，证明结构 **非噪声**。
- **Tier B**（原生 Valhalla，无 Transformer）：48 题 **25% acc**，数值子集 **8/9**；路线 **已跑通**，绝对能力 **未超 Transformer**。
- **KD 对照**（3B→0.5B）：**+22.92 pp** — 有监督范式上限；Valhalla 定位 **不同范式**，非 acc 单一竞赛。

**融资叙事**: 不卖「打败 GPT」；卖 **可拥有、可审计、可结构孵化的 AI 基础设施** + **Tier B 原生路线期权**。

## 1. 市场机会

### 1.1 痛点

| 痛点 | 规模信号 | Valhalla 切入 |

| API 税与锁定 | 企业 POC 依赖闭源 API，规模化单位经济崩溃 | 本地 Rust 运行时 + checkpoint |

## 2. 产品与解决方案

### 2.1 产品矩阵

| SKU | 描述 | Transformer 关系 |
|-----|------|------------------|
| **Hub Enterprise** | 四元路由 + Fate + dashboard | 可 **独立** 于 LLM 部署 |
| **Incubation Bridge (Tier A)** | export patch → 外部 0.5B–3B | **代理验证** 结构→行为 |
| **Native QA (Tier B)** | `valhalla_native_qa` 无 HF | **逐步替代** forward 依赖 |
| **StemCell / Tile 模块** | 结构生长引擎 | 非 Transformer 层 |
| **Full stack OEM** | Wave + Val-OS + Hub | 可启动主权 AI 工作站 |

### 2.2 价值主张（vs Transformer-only stack）

```
闭源 API 路径:     数据 → [黑箱 Transformer] → 答案 → 持续 API 费
传统私有化:        数据 → [自托管 Transformer + RAG/LoRA] → 答案 → GPU + 微调
Valhalla 路径:     语料 → [结构孵化 checkpoint] → Tier A patch 或 Tier B 原生 → 可审计演化
```

### 2.3 我们 **不** 卖什么

- 通用 benchmark SOTA（MMLU / HumanEval 榜首）
- 「已替代 GPT-4」类表述
- 无实验支撑的 +4.17pp 类增益（已作废）

## 3. 竞争与定位

### 3.1 竞争矩阵

| 对手类别 | 代表 | 他们强 | Valhalla 楔子 | 诚实差距 |
|----------|------|--------|---------------|----------|
| 闭源 LLM API | OpenAI, Anthropic | 通用 IQ | 主权、无 API 税 | 通用推理 |
| 云托管 AI | Bedrock, Azure OpenAI | SLA + 生态 | 客户自有二进制 | 弹性规模 |
| 编排框架 | LangChain, LlamaIndex | 插件生态 | **原生 runtime + 结构引擎** | OSS 社区 |
| 向量/RAG | Pinecone, pgvector | 成熟检索 | **结构写入 checkpoint** | 查询语言成熟 |
| 微调/KD | HF PEFT, Axolotl | acc 提升明确 | **无 backprop 孵化** | 当前 acc 更低 |

### 3.2 定位句

**中文**: Valhalla 是可私有化部署、路由可观测的 **AI 孵化运行时**——Tier A 证明结构能触达语言模型行为；Tier B 让 Valhalla 自身承担推理，面向不能接受 API 锁定与黑箱调度的政企与垂直行业。

**English**: Valhalla is an ownable, auditable **AI incubation runtime**—Tier A proves structure reaches model behavior; Tier B moves inference onto Valhalla for regulated and vertical teams who cannot accept API lock-in.

### 7.3 投资人 FAQ

**Q: 你们比 Transformer 强吗？**  
A: 在 48Q strict 上 **不强**——0.5B baseline ~69%，Valhalla Tier B ~25%。我们卖 **孵化范式与主权基础设施**，不是 benchmark 冠军。

**Q: 那 +4.17pp 呢？**  
A: **已作废**——宽松 MCQ 评分 bug；strict 下 Hub -10.42pp。

**Q: KD +22.92pp 是否说明 Valhalla 没必要？**  
A: KD 需要 **3B teacher + 梯度训练**；Valhalla 主张 **无 backprop 结构孵化**——不同范式，服务 **不能/upload 数据做 KD** 的客户。
