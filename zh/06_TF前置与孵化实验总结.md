## 4. 亮点

1. **Body 分离跑通**：hub / tile / stemcell / triad 各 3 协议，12 臂 E2E 完成（~3.4 min/臂量级）。
2. **Tile / StemCell / Triad（direct）均可 +1 题**：GSM_03 word problem 在累积 patch 后答对。
3. **LLM 前置在 tile/stem 上可与 direct 同效**：`tile_tf_front_cumulative`、`stemcell_tf_front_cumulative` 亦 58.33% — 与上次仅 triad 时 LLM 前置 0 pp **不同**，body 分离后 **tile/stem 路径下 LLM 前置有效**。
4. **无 strict 对→错倒退**：已对的 MATH_01–04、GSM_04、MCQ_PHYS_01 在进步臂上保持。
5. **无 collapse**：无空输出 / 全 `#` 塌缩。

## 5. 问题

1. **Hub 三臂 Final 全 0 pp**：Hub-only patch 未带来可保留增益；且 hub 累积臂 round 2 曾 58.33%，round 3 **回退** 至 50%。
2. **Triad + LLM 前置仍无效**：`triad_tf_front_cumulative` 全程 50% — 三路融合 + LLM 扩写在本 12 题无收益。
3. **fresh_reload 全灭**：4 body × fresh_reload 均 50% — reload 协议下 LLM 前置 **完全无作用**。
4. **MCQ 无 body 改善**：进步臂仍 **1/4 MCQ**（仅 PHYS_01）；GSM_01/02 仍错。
5. **Hub round 3 回退**：说明 **多轮累积 patch 不稳定**，后一轮可抹掉前一轮收益。
